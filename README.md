# Student Management System

A **Front-End Student Management System** built with **HTML, CSS, and JavaScript**.  
This project allows users to **signup, login, manage student records**, and view a **dashboard summary**.  
It features **dark mode, 3D hover animations**, and a clean, professional UI.
